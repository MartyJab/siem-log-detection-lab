# Blue Team Home Lab

A Docker-based Blue Team lab for learning log analysis, traffic monitoring, and basic red team path scanning.

## Features

- Simulated web traffic
- Reconnaissance simulation
- Apache access log reporting
- High traffic detection
- HTTP error rate detection
- Suspicious path detection

## Architecture

Client1 and Client2 --> Apache Web Server --> Apache Acces Logs --> Reporting and Detection  

## Technologies

- Docker
- Python
- Apache2
- Linux

## Project Goals

- Learn log analysis
- Understand detection logic
- Simulate attacker behavior
- Learn Docker
